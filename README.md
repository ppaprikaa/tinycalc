# TinyCalc

Simple interpreter I wrote to learn FSMs(DFAs, NFAs), tokenization and basics of ASTs.

```bash
git clone https://github.com/ppaprikaa/tinycalc.git
cd tinycalc
cargo build 
./target/debug/tinycalc
```

## Operators
+, -, *, /

### Precedense
*, / over +, -

## Grouping
with `( )`
